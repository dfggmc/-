下面是官方介绍(有修改）：

viaproxy
独立代理，允许玩家加入每个Minecraft服务器版本（经典，Alpha，Beta，发布，基岩）

支持的服务器版本：
经典（c0.0.15 - c0.30，包括 CPE)
阿尔法 （a1.0.15 - a1.2.6）
测试版 （b1.0 - b1.8.1）
版本 （1.0.0 - 1.20）
愚人节（3D共享软件，20w14无限）
战斗快照（战斗测试8c）
基岩版 1.20.0（开发中）
支持的客户端版本
版本 （1.7.2 - 1.20）
基岩版（需要间歇泉插件)
经典版、阿尔法版、测试版、1.0 - 1.6.4 版（仅限直通版）
ViaProxy 支持从任何列出的客户端版本加入任何列出的服务器版本。

特殊功能：
支持加入联机模式服务器
支持在从所有列出的客户端版本启用了聊天签名的服务器上加入。

如何使用
运行 jar 文件（使用 java -jar ViaProxy-whateverversion.jar --help）
查看可用的配置选项，并像在 GUI 中一样使用您需要的配置选项
使用 start 命令启动代理并测试它是否有效（使用服务器的公共地址和您配置的绑定端口加入）
玩得愉快！
下面是一个示例命令，允许玩家加入您的serverip：25568并连接到在端口25565上运行的beta 1.7.3服务器：java -jar ViaProxy.jar --bind_port 25568 --target_ip 127.0.0.1 --target_port 25565 --version b1.7-b1.7.3
